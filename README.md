## 改善できるところ
基本的にはクイズアプリとロジックは変わらないので、複数選択ができるように回答した項目をスコア値として管理できれば、尚いいかなと思いました。

- [ ] スコアを保存して回答結果の条件分岐ができれば面白そう。
- [ ] 料理アプリで、今日の献立を出力してくれたら楽しそう。
